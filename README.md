## Description

Event Management using Nest.js, Postgresql, Sequelize, Prettier and Swagger.

## Installation

```bash
$ npm install
```

## Setup environment
1. Create an .env file using the .env.sample format
2. Change all values in .env file with yours

## Code Format
```bash
$ npm run format
```

## Running the app

```bash

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
